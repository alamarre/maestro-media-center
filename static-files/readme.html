Maestro Media Center

Setup

  1. Move the files to a convenient place on your hard drive
  2. Ensure you have Java installed at version 7 or higher. If you're not sure download a JRE from http://www.oracle.com/technetwork/java/javase/downloads/index.html
  3. Setup your video folder by doing one of the following:
    • Copy the videos and folders you want into the videos folder in the Maestro folder
    • On Windows find the folders you want to add, right click on them and choose create shortcut and copy the shortcuts into the videos folder
    • Create symlinks to the folders you want in your videos folder. How to do this is beyond the scope of this readme.

Running

  1. Do one of the following to start the media center:
    • Double click the AllInOne-1.0.jar file
    • On Windows double click the run.bat file. This will give more information if it fails.
    • On Linux or Mac run the run.sh script. You may have to make it executable first. This will give more information if it fails.
    • From a command line run java -jar AllInOne-1.0.jar
  2. Go to http://localhost:8080 and enjoy!
